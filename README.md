## 🗃️ Supabase Setup

1. Create a free account at [supabase.com](https://supabase.com)
2. Create a new project
3. Open the **SQL Editor** and run this query:

```sql
create table applications (
  id bigint generated by default as identity primary key,
  company text not null,
  job_title text not null,
  job_type text not null,
  location text,
  application_date date not null,
  deadline date,
  status text not null,
  priority text not null,
  source text,
  job_description text,
  notes text,
  created_at timestamp with time zone default now(),
  updated_at timestamp with time zone default now()
);
```

4. Go to **Settings → API** and copy your **Project URL** and **anon public key**

## 💻 Local Setup

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/gradpath-ai.git
cd gradpath-ai
```

### 2. Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment variables

```bash
cp .env.example .env
# Edit .env with your Supabase URL, Supabase key, and (optionally) OpenAI key
```

### 5. Run the app

```bash
streamlit run app.py
```

## ☁️ Deployment (Streamlit Community Cloud)

1. Push your project to a public GitHub repo
2. Go to [share.streamlit.io](https://share.streamlit.io) and connect your repo
3. Set the main file to `app.py`
4. Under **Advanced settings → Secrets**, add:

```toml
SUPABASE_URL = "https://your-project.supabase.co"
SUPABASE_KEY = "your-anon-key"
OPENAI_API_KEY = "sk-..."   # Optional
```

5. Deploy — Streamlit Community Cloud is free for public repos

## 🤖 AI Feature Explanation

The **AI Job Preparation Assistant** on the AI Assistant page:

- Accepts a job description from an existing application or manual paste
- Calls OpenAI's `gpt-4o-mini` model with a carefully crafted prompt
- Returns: job summary, key skills, resume keywords, interview topics, follow-up email, next steps
- **Falls back** to a keyword-matching rule-based analyzer if no API key is set

This lets the app work fully without an OpenAI key while providing richer analysis when one is available.

---

## 📊 Data Science Feature Explanation

The **Analytics page** is the data science bonus feature:

- Uses `pandas` for groupby aggregations, date filtering, and computed metrics
- Uses `Plotly` for interactive, themed charts (donut, line, bar, scatter)
- Generates **Smart Insights**: best source, urgent deadlines, recent activity, no-response detection
- All charts use a consistent dark theme with a custom color palette
