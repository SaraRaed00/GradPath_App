# 🎓 GradPath AI — Job Application Tracker

> A Streamlit web app for university students and fresh graduates to manage job applications, track hiring progress, analyze job search trends, and prepare for interviews with AI.

---

## 🔗 Live App

https://gradpath-app.streamlit.app/



## ✨ Features

- **Dashboard** — KPI metrics, pipeline view, upcoming deadlines
- **Add Application** — Form with full validation for all fields
- **Applications** — Search, filter (status, type, priority), sort, edit, delete
- **Analytics** — Interactive Plotly charts + AI-generated smart insights *(Data Science bonus)*
- **AI Assistant** — Job description analyzer using OpenAI GPT-4o-mini *(AI bonus)*
- **Sample Data** — One-click demo data insertion for presentation

##  Supabase Setup

create table applications (
  id bigint generated by default as identity primary key,
  company text not null,
  job_title text not null,
  job_type text not null,
  location text,
  application_date date not null,
  deadline date,
  status text not null,
  priority text not null,
  source text,
  job_description text,
  notes text,
  created_at timestamp with time zone default now(),
  updated_at timestamp with time zone default now()
);



##  AI Feature Explanation

The **AI Job Preparation Assistant** on the AI Assistant page:

- Accepts a job description from an existing application or manual paste
- Calls OpenAI's `gpt-4o-mini` model with a carefully crafted prompt
- Returns: job summary, key skills, resume keywords, interview topics, follow-up email, next steps
- **Falls back** to a keyword-matching rule-based analyzer if no API key is set

This lets the app work fully without an OpenAI key while providing richer analysis when one is available.

---

##  Data Science Feature Explanation

The **Analytics page** is the data science bonus feature:

- Uses `pandas` for groupby aggregations, date filtering, and computed metrics
- Uses `Plotly` for interactive, themed charts (donut, line, bar, scatter)
- Generates **Smart Insights**: best source, urgent deadlines, recent activity, no-response detection
- All charts use a consistent dark theme with a custom color palette
